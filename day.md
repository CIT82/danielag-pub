# A Day in My Life
### By Daniela Gamez
1. **Morning**
- Begrudgingly get out of bed 
- Feed my two cats (who never fail to remind me)
- Eat breakfast 
- Do some quick bible reading
- Get dressed
- Go to ceramics class (Mon/Wed)
2. **Afternoon**
- Clean up workstation (tidy up wheel and tools, recycle unused clay, etc)
- Grab lunch 
- Wait for ride 
- Head home 
- Take a "little" break 
3. **Late Afternoon**
- Work on assignments
- Do whatever chores need to get done 
4. **Evening**
- Help Dad prepare dinner 
- Wait for Mom to get home
- Eat dinner 
- Work on assignments if I havent already done so 
- *Chill* (Do some sketching, play videogames with brother, or watch TV)
- Sleep
